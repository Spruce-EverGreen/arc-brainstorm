# Arc Brainstorm

A collaborative brainstorming canvas for visual project planning. Draw boxes, connect with arrows, add text, and export as JSON for Willow to ingest into specs.

## Quick Start

```bash
npm install
npm run dev
```

Then open `http://localhost:5173` on your 2-in-1 or any device.

## Features

- **Draw** boxes with drag/resize
- **Connect** boxes with directional arrows
- **Label** everything with text
- **Color** code for organization
- **Export** as JSON for spec generation

## Tools

- **Select** (↖) — Move, resize, select elements
- **Box** (▢) — Draw rectangles
- **Arrow** (→) — Connect boxes
- **Text** (T) — Add labels

## Keyboard

- `Delete` / `Backspace` — Delete selected element
- `Escape` — Deselect
- `Enter` — Confirm text label

## Export

Click **View JSON** to see the data structure. Click **Export** to download as `arc-brainstorm-{timestamp}.json`.

Share the JSON with Willow for spec generation.
